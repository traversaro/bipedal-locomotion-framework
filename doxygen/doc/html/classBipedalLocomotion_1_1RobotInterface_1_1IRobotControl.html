Parameters

controlModea control mode for all the joints.
Returns
True/False in case of success/failure.
Note
In case of position control the values has to be expressed in rad, in case of velocity control in rad/s. If the robot is controlled in torques, the desired joint values are expressed in Nm. Call this function if you want to control all the joint with the same control mode. Otherwise call setReferences(Eigen::Ref<const Eigen::VectorXd>, const std::vector<IRobotControl::ControlMode>&).
