Detailed Description

template<typename State, typename StateDerivative, typename Input>
class BipedalLocomotion::System::DynamicalSystem< State, StateDerivative, Input >

DynamicalSystem defines a continuous time dynamical system, i.e.

\(\dot{x}=f(x, u, t)\). Please inherit publicly from this class in order to define your custom dynamical system.

Template Parameters
Statetype used for describing the state (i.e. it has to be a std::tuple vectors/matrices or in general classes).
StateDerivativetype used for describing the state derivative (i.e. it has to be a std::tuple of vectors/matrices or in general classes).
Inputtype used for describing the input (i.e. it has to be a std::tuple of vectors/matrices or in general classes).
