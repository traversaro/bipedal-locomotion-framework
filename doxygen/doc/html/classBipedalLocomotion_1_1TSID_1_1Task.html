Detailed Description

Task describes a control problem element.

The element is described by a matrix \(A\) and a vector \(b\). This class describes both a linear equality constraint and a linear inequality constraint. In case of equality constraint \( A \) and \( b \) represents: \( Ax = b\) In case of inequality constraint \( A \) and \( b \) represents: \( Ax \le b \)

Note
Please inherit this class if you want to build your own optimal control problem.
